# Magic PWA 老師操作指南 (Teacher User Guide)

歡迎使用 Magic PWA 教學系統！本指南將協助老師快速熟悉系統操作，從登入、查看教材到管理課程內容。

---

## 1. 系統登入與身分驗證
根據系統流程圖，老師的存取路徑如下：
1. **進入入口**：訪問 `app.example.com`。
2. **登入驗證**：輸入您的帳號 (Username) 與密碼。系統使用 **Amazon Cognito** 進行安全驗證。
   - *注意：若為首次登入，系統可能會要求您修改初始密碼。*
3. **角色分流**：登入成功後，系統會自動識別您的 `teacher` 角色，並導向老師專屬主頁。

## 2. 老師主頁 (Teacher Main)
老師主頁提供與學生相似但權限更廣的介面：
- **查看所有教材**：老師可以點擊進入**所有的 Week 內容**，不受學生端的「解鎖日期」或「裝置限制」約束。
- **教材概覽**：以 Week 為單位顯示列表，方便您快速預覽教學內容。

## 3. 教材管理 (Management Panel)
作為老師，您可以點擊主頁上的 **「教材管理」** 按鈕進入後台設定：

### A. 管理 Week 內容
- **查看現有 Week**：列表中會顯示目前已建立的所有課程週次。
- **建立或編輯 Week**：
  - 您可以指定 **Week 編號** (如 Week 1, Week 2)。
  - 設定 **標題** (Title) 與 **描述** (Description)。

### B. 上傳與關聯教材 (S3 Content)
- **PDF 講義**：您可以為每個 Week 指定 PDF 檔案路徑。系統會從私有的 S3 Bucket 安全地讀取內容。
- **魔術互動頁面 (Magic HTML)**：
  - 系統會從 S3 自動抓取可用的魔術網頁資產 (如：無敵預言術、奧術讀心等)。
  - 您可以為每個 Week **勾選最多 8 個魔術頁面**，這些頁面將出現在學生的 Week 詳情頁中。

## 4. 課程詳情與教學 (Week Detail)
點擊特定的 Week 進入詳情頁後：
- **PDF 預覽器**：直接在瀏覽器中開啟該週的教學講義。
- **互動魔術**：點擊下方已指派的魔術卡片，即可開啟對應的 HTML 互動頁面。
- **安全性**：所有的資源都透過短效授權 URL (Pre-signed URL) 提供，確保教材不會被非法外流。

## 5. 常見問題與限制
- **存取拒絕 (Deny Access)**：若您的 Token 過期或權限異常，系統會導向拒絕存取頁面。請重新登入以刷新權限。
- **同步延遲**：若剛上傳的教材未出現，請嘗試重新整理頁面。
